# Criar um repositório GIT no GITHub

## Primeiro Passo 
> Instalar o Git

## Segundo Passo
1. Crie uma pasta para abrigar os arquivos do seu projeto

> mkdir nome-da-pasta

2. Entre na pasta que você criou

> cd nome-da-pasta

3. Inicializar o seu repositorio Git

> git init
